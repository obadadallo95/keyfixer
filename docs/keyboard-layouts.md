# Keyboard Layout Specifications

KeyFixer maps physical key positions between standard English QWERTY layouts and Arabic input layouts.

## Special Characters & Ligatures

- **Ligature `لا` (`b`)**: Maps to key `b`.
- **Ligature `لأ` (`G`)**: Maps to `Shift + g`.
- **Ligature `لإ` (`T`)**: Maps to `Shift + t`.
- **Ligature `لآ` (`B`)**: Maps to `Shift + b`.

## Diacritics (Tashkeel)

- `Q` → `َ` (Fatha)
- `W` → `ً` (Tanween Fath)
- `E` → `ُ` (Damma)
- `R` → `ٌ` (Tanween Damm)
- `A` → `ِ` (Kasra)
- `S` → `ٍ` (Tanween Kasr)
- `X` → `ْ` (Sukun)
- `Z` → `~` (Shadda)

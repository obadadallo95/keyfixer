# Product Roadmap

Future planned features and enhancements for KeyFixer.

## Short-term (v1.1.0)
- [ ] Extension global hotkey shortcut (e.g. `Alt + Shift + K`) to fix selection instantly.
- [ ] Keyboard shortcuts in Web App (`Ctrl + Enter` to copy, `Esc` to clear).
- [ ] Optional popup UI language toggle (Arabic / English UI).

## Mid-term (v1.2.0)
- [ ] Additional layout variants (e.g. French AZERTY / Arabic layout).
- [ ] Custom key mapping editor for specialized keyboard layouts.
- [ ] Optional local-only recent conversion history buffer in web app.

## Long-term (v2.0.0)
- [ ] Electron / Tauri lightweight cross-platform desktop background daemon with global hotkeys.
